# project-clockwise-r
Project Clockwise R(ChronoR) เป็นเกมแนว Rhythm กดตามจังหวะเพลงโดยใช้เข็มนาฬิกาเป็นหลัก ซึ่งได้ Concept หลัก ๆ มาจาก Puzzles ของ ZZZ และทำต่อยอดเป็นเกม
Gameplay : มีนาฬิกาอยู้กลางจอ 1-2 อัน แต่ละอันมี 2 เลนที่วางโน๊ตหลักๆ อยู่ได้แก่
Single Note
เมื่อเข็มทับกับโน๊ตตัวนี้ให้กดปุ่มใดก็ได้ใน keyboard ตามจังหวะเพลง
็Hold Note
กดเหมือนกับ Single Note แต่ต้องกดค้างแล้วปล่อยตรงจุดที่เป็นหางโน๊ต
Reverse Note
โน๊ต gimmick ที่กลับทิศทางการหมุนของเข็มนาฬิกา
Overclock/Overheat Note
โน๊ต gimmick ที่เพิ่ม/ลดความเร็วในการหมุนของเข็มนาฬิกา
Anormaly Note
โน๊ต gimmick เปลี่ยน bpm ของด่าน
